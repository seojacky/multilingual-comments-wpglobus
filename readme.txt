== Description ==
WPGlobus Multilingual Comments  is a plugin designed to provide multilingual commenting on your WordPress site when using the popular WPGlobus plugin. This plugin allows visitors to leave comments in different languages, making interaction with your content more convenient and accessible to a wider audience.
The plugin automatically filters comments based on the language of the current post, adds a language selection field to the comment form, saves the selected language when submitting a comment, and provides the ability to mass-edit comments in different languages. All of these features help in creating a multilingual community on your site and enhance the user experience.
WPGlobus Multilingual Comments  is a simple and effective solution for those looking to expand the multilingual capabilities of their WordPress site using the WPGlobus plugin.
